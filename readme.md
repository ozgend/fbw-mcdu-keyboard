## keyboard inputs for flybywire remote mcdu display

provides keyboard inputs for flybywire remote mcdu display on browser.

1. open remote mcdu display on browser (http://localhost:8380/interfaces/mcdu)
2. inject `key.js` into the page
3. good to go

### keyboard - mcdu key mapping

| keyboard    | mcdu key  |
| ----------- | --------- |
| `Tab`       | DIR       |
| `Insert`    | PROG      |
| `Home`      | PERF      |
| `PageUp`    | INIT      |
| `Enter`     | DATA      |
| `Delete`    | FPLN      |
| `End`       | RAD NAV   |
| `PageDown`  | FUEL PRED |
| `Esc`       | MENU      |
| `Shift`     | AIRPORT   |
| `F1`-`F6`   | LSK1-LSK6 |
| `F7`-`F12`  | RSK1-RSK6 |
| `Left`      | PREVPAGE  |
| `Right`     | NEXTPAGE  |
| `Up`        | UP        |
| `Down`      | DOWN      |
| `Backspace` | CLR       |
| `Space`     | SP        |
| `.`         | DOT       |
| `+` `-`     | PLUSMINUS |
| `/`         | DIV       |
| `*`         | OVFY      |
| `A-Z` `0-9` | A-Z 0-9   |
